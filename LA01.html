Basics of Linear Systems

When beginning to learn linear algebra, you must first become familiar with the idea of linear systems. Linear systems are one of the most central and important concepts in linear algebra. A linear system is defined as a set of multiple linear equations that use the same variables.

Linear equations take the form a1x1 + a2x2 + ... + anxn = b where a and b are always known constants, and x1 through xn are unknowns.

An example of a linear equation is: 

 7x1 + 11x2 + 5x3 = 54

An example of a linear system is: 

 4x1 + 7x2 + 5x3 = 23 
 1x1 + 2x2 + 9x3 = 11 
 8x1 + 6x2 + 3x3 = 42

Solution sets of Linear Systems

For any given linear system, the system could have
1) No solution
2) One solution
3) Infinitely many solutions

If a linear system has at least one solution, we call it consistent. If there is no solution to a system, it is inconsistent.

Example: Consider the following linear system 

 2x1 + 10x2 = 20 
 5x1 + 5x2 = 10

The solution of a linear system of equations is the list of all x values that make the system true when those x values are substituted in.
In this case, the solution of this system is x1 = 0, and x2 = 2. Or, you could also say (0,2)
We can verify this is the case by substituting 0 back in for x1, and 2 back in for x2. 

 2(0) + 10(2) = 20  ---> 20 = 20 
 5(0) + 5(2) = 10 ---> 10 = 10

As we can see, when these values are substituted in, they make both equations true. So, the solution of this system is (0,2). This particular system has only one solution, however, other systems can have either no solution or infinitely many solutions.

Coefficient and Augmented Matricies

Linear systems are most often written as coefficient and augmented matricies. A coefficient matrix is one that places only the variable coefficients from the equations in a matrix. An augmented matrix on the other hand is one that places both the variable coefficients from the equations AND the numbers they are equal to in a matrix.
Using the same example system from above, it's coefficient matrix would look like: 

 4X1 + 7x2 + 5x3 = 23           | 4   7   5 |                               
 1X1 + 2x2 + 9x3 = 11     ----> | 1   2   9 |                              
 8X1 + 6x2 + 3x3 = 42           | 8   6   3 |

For the above coefficient matrix, we are only using the coefficients of the variables. The first row in the system is 4x1 + 7x2 + 5x3, so we use 4, 7, and 5 for the first row of our coefficient matrix. The same process can be repeated for the other two rows.

And it's augmented matrix would look like: 

 4X1 + 7x2 + 5x3 = 23           | 4   7   5 | 23 |                             
 1X1 + 2x2 + 9x3 = 11     ----> | 1   2   9 | 11 |                            
 8X1 + 6x2 + 3x3 = 42           | 8   6   3 | 42 |

For the augmented matrix, notice how we also include the value that the numbers are equal to. The first row of our original system is 4x1 + 7x2 + 5x3 = 23, and since we are including the numbers they are equal to, we would use 4, 7, 5, AND 23. The same process can be repeated to obtain the rest of the matrix. One final thing to keep in mind is that for an augmented matrix the bar is placed between the variable coefficients and the number they are equal to because this is how matricies are set up to be solved.
